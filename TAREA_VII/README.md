- EXPLICACION DEL PROGRAMA
Este programa en C++ permite ingresar números enteros y clasificarlos automáticamente en tres estructuras dinámicas:

* Pila (Stack): almacena números impares
* Cola (Queue): almacena números pares
* Lista enlazada: almacena números negativos (excepto -1)

El sistema utiliza memoria dinámica mediante nodos, lo que permite almacenar datos sin límite fijo.

Además, el programa incluye un menú interactivo que permite al usuario realizar diferentes operaciones sobre las estructuras, como:

* Insertar números
* Mostrar contenido de cada estructura
* Eliminar elementos (pop y dequeue)
* Consultar el elemento superior o frontal (peek)
* Ordenar la lista de números negativos
* Mostrar estadísticas (cantidad de pares e impares)

- ¿CÓMO EJECUTARLO?
1. El usuario selecciona la opción “Ingresar números”
2. Ingresa valores uno por uno
3. El programa clasifica automáticamente:
   - Negativos → Lista
   - Pares → Cola
   - Impares → Pila
* El usuario puede interactuar con el menú para visualizar o modificar las estructuras

* El valor -1 se usa como indicador para dejar de ingresar números.
