# Tarea Hash CSV

## Descripción
Proyecto que implementa carga masiva de estudiantes desde un archivo CSV utilizando:

- C++: Tabla Hash manual con Separate Chaining.
- Java: HashMap.

Funciones:
- Cargar CSV
- Buscar estudiante
- Eliminar estudiante
- Mostrar tabla/estudiantes
- Mostrar estadísticas

## Ejecutar C++

Entrar a la carpeta:

cd cpp-hash

Compilar:

g++ main.cpp -o hash

Ejecutar:

.\hash

## Ejecutar Java

Entrar a la carpeta:

cd java-hashmap

Compilar:

javac Estudiante.java Main.java

Ejecutar:

java Main
